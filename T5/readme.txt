-Simplemente descomprimir archivo.zip y agregar los archivos incluidos en alguna carpeta del directorio de matlab.
Ejecutar 'Run' de archivo Neut_MontesT5.m. 

-Se incluye una base de datos de las features extraidas llamada 'features_extracted.mat'
para hacer la corrección más rápida. 

-Es importante mencionar que en algunas versiones de Matlab no se permite la declaración de funciones en un mismo script, sino
que se debe efectuar en otro. 
Para la versión mia no tiene problema (Matlab R2016b), y según investigué, tampoco debería haber problema
con las versiones que tienen ustedes (ayudante).
